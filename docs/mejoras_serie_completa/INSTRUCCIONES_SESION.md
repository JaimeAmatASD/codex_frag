# La serie de mejoras — cómo usarla

1. Descomprimí este zip en la raíz de codex_frag. Agrega cinco mejoras y el
   análisis en docs/. No pisa nada tuyo.

2. Las mejoras van EN ORDEN (01 → 05), una por vez. Terminá, commiteá y probá
   cada una en el Taller antes de arrancar la siguiente. La 04 es un EXPERIMENTO
   con veredicto tuyo escrito; la 05 es la más grande y va última.

3. Mensaje para pegarle a Claude Code al arrancar cada mejora (cambiá el número):

---
Leé docs/MEJORA_01_TENSION.md completo y docs/ANALISIS_BRAINSTORMING_JUL2026.md.
Mirá el código actual que el prompt indica. Seguí sus instrucciones: empezá
mostrándome los modelos y plantillas para mi visto bueno, y recién después
escribí el resto. Commits chicos con mensajes en castellano como los del repo.
---
